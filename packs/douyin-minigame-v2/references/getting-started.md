# 从零开始创建抖音小游戏

## 前置条件

- 营业执照（企业）或身份证（个人）
- 软件著作权登记证书（**必须是国家版权局发放的**，电子软著可准入但不可备案）
- 一台 Windows 或 macOS 电脑

## 第一步：注册入驻

1. 访问 [抖音开放平台](https://developer.open-douyin.com/) 注册账号
2. 完成**主体认证**：
   - 企业：提交营业执照 + 法人信息 → 完成对公验证
   - 个人：提交身份证信息
3. 主体认证通过后才能创建小游戏

> 注意：个人开发者也可以开发小游戏，但部分能力（如 IAP）可能有限制。

## 第二步：创建小游戏

1. 登录 [开发者控制台](https://developer.open-douyin.com/console)
2. 点击「创建小游戏」，填写基本信息
3. 进入「设置 → 基础设置」完善以下必填项：

| 必填项 | 说明 |
|--------|------|
| 名称 | 游戏名称，每年限改 2 次 |
| 头像 | 游戏图标 |
| 简介 | 一句话描述 |
| 详细介绍 | 游戏详细描述 |
| 软件著作权登记证书 | 必须提供，每年限改 2 次 |
| 版号（如有）| 有广电总局版号批文的填写 |
| IP 资质（如需）| 使用第三方 IP 需提供授权 |

4. 完成**自审自查报告**确认（2024年7月起可直接在控制台勾选，无需上传文件）
5. 设置**隐私政策**和**适龄提示**
6. 记下你的 **AppID**（在「开发管理 → 开发设置」中查看）

### 基础信息审核状态

| 状态 | 含义 |
|------|------|
| 审核中 | 信息已提交，等待审核 |
| 审核失败 | 需根据原因修改后重新提交 |
| 部分失败 | 部分项目通过部分失败，需修改失败项 |
| 审核通过待生效 | 审核已过但未在线上生效 |
| 已完成 | 所有字段审核通过且已生效 |

## 第三步：搭建开发环境

### 下载开发者工具

前往 [开发者工具下载页](https://developer.open-douyin.com/docs/resource/zh-CN/mini-game/develop/developer-instrument/developer-instrument-update-and-download) 下载。

### 创建工程

1. 打开开发者工具，登录（手机号/邮箱/抖音扫码）
2. 点击「创建一个小程序」
3. 填写：
   - **项目类型**：选择「小游戏」
   - **工程目录**：本地文件夹路径
   - **工程名称**：任意
   - **AppID**：填入第二步获取的 AppID

### 项目基础文件

无引擎项目需要包含以下文件：

**project.config.json** — 项目配置：
```json
{
  "setting": {
    "urlCheck": true
  },
  "appid": "你的AppID"
}
```

**game.json** — 游戏配置：
```json
{
  "deviceOrientation": "portrait",
  "showStatusBar": false
}
```

**game.js** — 游戏入口：
```javascript
const canvas = tt.createCanvas();
const ctx = canvas.getContext('2d');

ctx.fillStyle = '#ffffff';
ctx.fillRect(0, 0, canvas.width, canvas.height);
ctx.fillStyle = '#000000';
ctx.font = '40px Arial';
ctx.fillText('Hello 抖音小游戏!', 50, 100);
```

### 使用游戏引擎

| 引擎 | 操作 |
|------|------|
| **Cocos Creator** | 发布时选择「小游戏」→ 填入 AppID → 构建 → 用开发者工具打开输出目录 |
| **Laya** | 发布设置选择「字节跳动小游戏」→ 构建 → 用开发者工具打开 |
| **Unity** | 使用 Unity 小游戏转换工具 → 导出 → 用开发者工具上传 |

## 第四步：预览和调试

| 方式 | 操作 | 适用场景 |
|------|------|---------|
| IDE 预览 | 开发者工具内左侧窗口 | 快速验证逻辑 |
| 真机预览 | 点击「预览」→ 抖音/头条扫码 | 验证真实设备表现 |

## 第五步：备案

- 有广电总局版号批文 → 无需重复备案
- 无版号 → 基础信息审核通过 + 版本提审通过后，平台自动申报
- 进度查看：「设置 → 基础设置 → 资质信息 → 小游戏备案申请」

> 注意：电子软著可用于资质准入，但**暂不支持用于备案**，需先申请国家版权局发放的纸质软著。
